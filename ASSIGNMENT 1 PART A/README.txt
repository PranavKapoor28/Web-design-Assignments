# Website on Playstation games website

This website is a game and console sales website for playstation. Users can have a look at the top games available on the website and also find the price of latest playstation consoles and games. Apart from that, users can also contact the website owners if they have any doubt by writing their query in the Contact form.

# Instructions

1. Install and open VS_CODE.
2. Import the project and run main.html
3. Open main.html on new tab on chrome or any other browser.

# Tags used

1. Head -- The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.
2. Header -- The <header> tag in HTML is used to define the header for a document.
3. Body -- The <body> element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
4. Div -- <div> is used to divide the webpage into different parts and connect them by assigning them an id or class name.
5. Table -- <table> is used to add a table to the webpage.
6. Title -- The <title> element defines the title of the document. 
7. h1 -- <h1> tag is used to give the biggest heading to the webpage
8. Form -- <form> tag is used to add a form to the webpage with a submit button.
9. link -- The <link> tag defines the relationship between the current document and an external resource. The <link> tag is most often used to link to external style sheets or to add a favicon to your website.
10. a -- <a> is used to add a hyperlink to the webpage and connect to different websites or part of a webpage.
11. Button -- <button> tag is used to add a button and connect different pages in the website.
12. img -- <img> tag is used in the webpage to add various images of games and consoles.

The website includes favicon, hyperlink, image, button, table, form.
